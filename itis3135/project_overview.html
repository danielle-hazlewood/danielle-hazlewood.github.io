Overview

Describe Proposed Application and Purpose

This website is for Hair by Lily Darr, who is an upcoming hair styles that services the Lake Norman area. This website allows for clients to learn about the stylist, see pictures of work that has been done, and a way to send an inquires to Lily.

Purpose

The purpose of the website is to create a simple and streamline place for the clients to see if they may be interested in booking an appointment.
